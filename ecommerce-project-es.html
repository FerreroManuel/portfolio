Website con ecommerce usando Django

Información del proyecto

  • Categoría: Website
  • Cliente: MF! Soluciones Informáticas
  • Fecha de inicio del proyecto: Octubre 2022
  • Estado del proyecto: En desarrollo
  • Lenguages utilizados: Python, HTML, CSS y JavaScript

El proyecto consiste en un sitio web genérico para utilizar en una tienda, mediante el cual los clientes pueden hacer compras online. El mismo está hecho utilizando la librería Django de Python.
En él, los usuarios pueden crear una cuenta de usuario, agregar productos a su 'carrito' y a su lista de favoritos, realizar el pago online de manera segura (a través del SDK que provee Mercadopago), modificar sus datos de usuario y domicilios, reestablecer la contraseña en caso de olvido y eliminar su cuenta de usuario.
