Website with ecommerce using Django

Project information

  • Category: Website
  • Client: MF! Soluciones Informáticas
  • Project initial date: October 2022
  • Project status: In development
  • Languages: Python, HTML, CSS and JavaScript

The project consists of a store website, through which clients can make purchases online. It is made making use of Python Django library.
In it, users can create a user account, add products to their 'basket' and their wishlist, make the secure payment online (through Mercadopago's SDK), edit their user information and addresses, reset their password if forgotten and delete their account.
