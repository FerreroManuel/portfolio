Morella v2 (c/GUI)

Información del proyecto

  • Categoría: Desarrollo de software
  • Cliente: Bicon S.A.
  • Fecha de inicio del proyecto: Agosto 2022
  • Estado del proyecto: Pausado
  • Lenguages utilizados: Python y SQL

Morella es un sistema de gestión desarrollado para cubrir las necesidades de Bicon S.A., una empresa ubicada en la ciudad de Rosario dedicada a la administración de panteones sociales.
Luego de haber sido presentada la versión simple, que utiliza la consola de Windows como interfaz gráfica, se decidió comenzar con un proyecto de mejora. Este proyecto abarca tanto la interfaz gráfica, la cual está siendo desarrollada mediante la librería PySide6 para Python, como el back-end, mejorando procesos para poder lograr un mayor rendimiento del sistema.
Al igual que con la versión sin GUI, con Morella v2 se podrán realizar las cobranzas, el registro de la caja diaria, la revisión de los estados de cuenta de los asociados, y la comunicación con estos pero mediante una interfaz más actualizada y un rendimiento mayor.
